# 飞机模拟（气流/失速/风切变）—网页版

直接双击打开 `index.html` 就能玩（离线可运行）。

## 操作
- W / S：升降舵（俯仰）
- A / D：副翼（滚转；会导致升力损失，更像“硬拉会掉高度”）
- ↑ / ↓：油门
- Q / E：配平（Trim，持续性微调俯仰）
- Space：重置

## 你会感受到的“更模拟”的点
- **AoA（迎角）驱动的升力/阻力**：不是简单往上推，而是基于相对气流方向计算。
- **失速曲线**：迎角过大时升力衰减（同时阻力变大），低速大仰角会明显“掉下去”。
- **风切变**：越接近地面风越小，高空风更强；穿过不同高度会有手感变化。
- **热气流/下沉气流**：像气泡一样的气团，上升/下沉能把你抬起来或按下去。
- **阵风 + 乱流**：会推你、会抖你，配合配平更像“在空气里飞”。

## 小技巧
- 低速时不要一直猛拉 W：看 HUD 的 AoA，接近 15° 就容易失速。
- 遇到上升气流（黄色圈）可以略收油 + 小仰角，能“蹭”高度。
- 接近地面时风变小，油门/配平会更好控，但也更容易因为速度低而失速砸地。
